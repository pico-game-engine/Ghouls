## Ghouls
A round-based survival game built with the Pico Game Engine. Search for safety and weapons during the day and fight off ghouls at night. Explore the world, gather resources, and survive as long as you can!

### Features
- Weapons: Find and use a variety of weapons (shooters and launchers) to fend off the ghouls.
- Day/Night Cycle: Round-based survival gameplay with a day/night cycle. During the day, players can explore, gather resources, and prepare for the night. At night, players must defend themselves against waves of ghouls.
- Online Multiplayer: Team up with friends or other players online to survive together. (Coming Soon)

### Controls
- Up/Down/Left/Right: Move
- Center Button: Fire Weapon (if equipped)
- Back Button: Open/Close Game Menu

### Installation
This project has two configuration files: `config.hpp` and `engine_config.hpp`. The former is for game-specific settings, while the latter is for engine-specific settings. Both files are located in the `src` directory and are required to be configured before building the game (otherwise you'll encounter compilation errors). More details can be found [here](Configuration.md). Set your desired settings in both files, then compile and run the game!
